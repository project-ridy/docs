# Ridy — 와이어프레임

## 0. 랜딩 페이지

```
┌──────────────────────────────────────┐
│ Ridy                    [시작하기]    │
│──────────────────────────────────────│
│ 함께 타는 출퇴근, 더 가벼운 하루       │
│ 같은 회사 동료와 안전하게 매칭         │
│ [초대 코드로 시작] [서비스 보기]       │
│                                      │
│ 평점 4.8  운행 12,400회  CO₂ 8.2t↓   │
│──────────────────────────────────────│
│ 카풀 매칭 │ 자동 정산 │ 친환경 임팩트  │
│──────────────────────────────────────│
│ 1 가입 → 2 매칭 → 3 탑승             │
│──────────────────────────────────────│
│ 오늘 출근길부터 함께 타세요            │
│ [Ridy 시작하기]                       │
│ 푸터: 서비스/보안/문의                │
└──────────────────────────────────────┘
```

## 1. 온보딩 & 인증

### 초대 코드 입력 → 소셜 로그인
```
┌─────────────────┐
│                  │
│    🚗 Ridy      │
│  함께 타는 출퇴근│
│                  │
│ 초대 코드: ______│
│                  │
│ [카카오 로그인]   │
│ [구글 로그인]    │
│                  │
│  초대 코드로만   │
│  가입 가능합니다  │
└─────────────────┘
```

### 프로필 설정 (신규 사원)
```
┌─────────────────┐
│  프로필 설정     │
│                  │
│  [프로필 사진]   │
│  이름: _______   │
│  사번: _______   │
│  연락처: ______  │
│                  │
│  🚗 차주로 시작  │ (토글)
│  차량 정보 입력   │
│                  │
│  [시작하기]      │
└─────────────────┘
```
