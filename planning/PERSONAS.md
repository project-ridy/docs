# Ridy — 유저 페르소나

## 페르소나 1: 사원 김도윤 (28세)

- **소속**: 테크스타터 개발팀 사원
- **거주지**: 수원
- **통근**: 매일 수원→강남 사옥, 약 1시간 20분
- **고통**: 주차비 월 30만원, 혼자 운전하면 지루하고 비싸
- **니즈**: 같은 회사 동료와 카풀해서 비용 절반 분담, 사내 검증된 사람이라 안심
- **기술 수준**: 스마트폰 능숙, 사내 앱/슬랙 사용

## 페르소나 2: 신입 사원 이하늘 (24세)

- **소속**: 테크스타터 마케팅팀 신입
- **거주지**: 송파구 잠실
- **통근**: 버스 2번 환승, 약 50분
- **고통**: 아침 버스 만원, 환승할 때마다 땀, 차 있는 동료와 카풀하고 싶지만 누구한테 물어볼지 모름
- **니즈**: 사내 카풀 앱에서 방향 맞는 동료 쉽게 찾기, 정기 카풀로 매일 같은 팀원과
- **기술 수준**: 앱 사용 매우 능숙

## 페르소나 3: 차주 박준서 (35세)

- **소속**: 테크스타터 개발팀 시니어
- **거주지**: 분당
- **통근**: 자차 출퇴근, 약 30분
- **고통**: 뒷좌석 3개가 항상 빈 상태, 유류비 부담
- **니즈**: 같은 회사 동료만 태워서 안심, 유류비 보전, 사내 커뮤니티로 소속감
- **기술 수준**: 평범, 보안/안전에 민감

## 페르소나 4: 기업 관리자 최서연 (38세)

- **직업**: 테크스타터 HR 팀장
- **고통**: 사원 출퇴근 교통비 지원 부담, ESG 경영 압력, 사내 커뮤니티 약화
- **니즈**: 사내 카풀 플랫폼 도입 → 비용 절감 + ESG 실적 + 조직 문화 개선
- **기술 수준**: 관리자 대시보드 사용 가능, 초대 코드 발급/통계 확인

## 핵심 가정

- **회사 단위 폐쇄 서비스**: 같은 회사 사원끼리만 카풀 가능
- **초대 코드 기반 가입**: 기업 관리자가 발급한 초대 코드로만 가입
- **사내 검증**: 같은 회사 소속이라 신뢰도가 높음 (별도 신원 인증 불필요)
